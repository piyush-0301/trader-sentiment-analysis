# Trader Performance vs Market Sentiment (Hyperliquid Analysis)

This repository contains the completed Round-0 Assignment for the Data Science / Analytics Intern role at Primetrade.ai. It analyzes how crypto market sentiment (Fear/Greed) shapes trader behavior, psychology, and structural profitability.

## 📁 Repository Structure
* `/notebooks`: Contains the core data cleaning, alignment, feature engineering, and exploratory analysis pipeline.
* `/images`: Visual plots used to back data insights.
* `app.py`: Interactive Streamlit dashboard script for exploring results.
* `requirements.txt`: Python dependencies required to reproduce the environment.
